Возможно я не полностью разобрался в том как правильно работать с тестами, но далее опишу как запускать именно эти тесты.
Для проверки модульный тестов обязательно указывайте перед командой запуска сервера PORT=80 BUG_ID=<номер бага> (Например: "PORT=80 BUG_ID=1 npm run start"),
а сами тесты запускайте командой "npm run test:unit. Модульные тесты покрывают BUG_ID=1-3.
Для запуска интеграционных тестов также перед запуском сервера указывайте BUG_ID (Например: "BUG_ID=9 npm run start"). 
А потом уже запускайте hermione в gui или без. Интеграционные тесты покрывают BUG_ID=4-10.
